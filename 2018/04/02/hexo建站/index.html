利用的HEXO建站

  1. 需要安装node.js和git；傻瓜式安装即可。
  2. 注册GitHub账号。
  3. 建立一个文件夹，在里面右击git Bash Here。

  4. 安装hexo-cli

    npm install hexo-cli -g

    -g是代表全局安装
    检查hexo-cli是否安装成功

    hexo -v

  5. 初始化文件夹

    hexo init Hexo

    初始化好后 关闭git 然后Hexo 右击git Bash Here

    cd hexo  //进入文件夹  Hexo是你初始化时的文件名
cd /  返回根目录
cd ..  //返回上一级目录

  6. hexo g  生成静态的html文件。

7.
hexo s 开启本地服务器 在浏览器中输入http://localhost:4000/

  1. hexo new 你博文的名字  你可以在主目录下的source里面的_posts文件夹里面可以看到将里面的

    改成

    deploy：
  type： git
  repo: https://github.com/longlog/longlog.github.io.git //仓库名要和GitHub账号名一致
  2. 将文件上传到GitHub上面，要先hexo g 再hexo d进行提交，完成后可以用 yourname.github.io进行访问
